CalendarOnline :

The main class which was alone at first. We removed the part which handle the connection that is now able on Calendar Builder.

CalendarBuilder :
This class handle only connection to the cloud server. We removed the part which read a file from a path on UserCredentials.

UserCredentials :
This class read from a specific path, information about the connection to the cloud server.

Information to give to connect :

username (in file)

password (in file)

calendarId (in file)

url (in file ?)

posturl (something needed only with Nextcloud's server and empty with Fruux)
